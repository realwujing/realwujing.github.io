    • ж ёеҝғжҖқжғі: йҡҸзқҖзҪ‘з»ңе®ҪеәҰ \(n_l\) и¶Ӣеҗ‘ж— з©·еӨ§пјҢжҲ‘д»¬йңҖиҰҒж»Ўи¶ідёӨдёӘжқЎд»¶д»ҘдҝқжҢҒи®ӯз»ғзЁіе®ҡпјҡ
      1. A1 (Activation): еҲқе§ӢеҢ–ж—¶зҡ„жҝҖжҙ»еҖјеә”дҝқжҢҒ \(O(1)\)гҖӮ
      2. A2 (Update): з»ҸиҝҮдёҖж¬ЎжўҜеәҰжӣҙж–°еҗҺпјҢжҝҖжҙ»еҖјзҡ„еҸҳеҢ–йҮҸеә”дҝқжҢҒ \(O(1)\)гҖӮ

    Page 9: жҺЁеҜј muP (жқЎд»¶ A1)¶

    • еңәжҷҜ: з®ҖеҚ•зҡ„ж·ұеәҰзәҝжҖ§зҪ‘з»ң \(h_l = W_l h_{l-1}\)гҖӮ
    • жҺЁеҜј: еҰӮжһң \(W_l \sim N(0, \sigma^2)\)пјҢдёәдәҶи®©иҫ“еҮәиҢғж•°дёҚйҡҸе®ҪеәҰ \(n_l\) зҲҶзӮёпјҢжҲ‘д»¬йңҖиҰҒи®ҫзҪ®ж–№е·® \(\sigma \approx 1/\sqrt{n_l}\)гҖӮиҝҷдёҺ Xavier еҲқе§ӢеҢ–дёҖиҮҙгҖӮ

    Page 10: жҺЁеҜј muP (жқЎд»¶ A2)¶

    • еңәжҷҜ: иҖғиҷ‘жқғйҮҚжӣҙж–° \(\Delta W_l = -\eta \nabla \ell\)гҖӮ
    • жҺЁеҜј: жӣҙж–°еҗҺзҡ„жҝҖжҙ»еҖјеҸҳеҢ– \(\Delta h_l = W_l \Delta h_{l-1} + \Delta W_l h_{l-1}\)гҖӮжҲ‘д»¬еёҢжңӣиҝҷдёҖйЎ№д№ҹжҳҜ \(O(\sqrt{n_l})\)пјҲдёҺжҝҖжҙ»еҖјеҗҢзә§пјүгҖӮ

    Page 11: жҺЁеҜј muP (жқЎд»¶ A2) Part 2¶

    • з»“и®ә: дёәдәҶж»Ўи¶ідёҠиҝ°жқЎд»¶пјҢеӯҰд№ зҺҮ \(\eta\) еҝ…йЎ»йҡҸзқҖе®ҪеәҰ \(n_l\) зҡ„еўһеҠ иҖҢеҮҸе°ҸпјҢе…·дҪ“жқҘиҜҙ \(\eta_l = \Theta(1/n_{l-1})\)гҖӮ
    • ж„Ҹд№ү: иҝҷе°ұжҳҜдёәд»Җд№ҲеңЁеӨ§жЁЎеһӢи®ӯз»ғдёӯпјҢйҖҡеёёжЁЎеһӢи¶Ҡе®ҪпјҢеӯҰд№ зҺҮи¶Ҡе°Ҹзҡ„зҗҶи®әдҫқжҚ®гҖӮ

    Page 12: muP е°Ҹз»“¶

    • жҖ»з»“: muP жҳҜдёҖеҘ—е…ідәҺеҲқе§ӢеҢ–е’ҢеӯҰд№ зҺҮзҡ„зј©ж”ҫи§„еҲҷпјҲдҪңдёәе®ҪеәҰзҡ„еҮҪж•°пјүгҖӮ
    • Initialization: \(\Theta(1/\sqrt{n})\)
    • Learning Rate: \(\Theta(1/n)\) (еҜ№дәҺ Adam)
    • еҜ№жҜ”: ж ҮеҮҶеҸӮж•°еҢ–пјҲStandardпјүйҖҡеёёеҸӘеҒҡеҲқе§ӢеҢ–зј©ж”ҫпјҢиҖҢеҝҪз•ҘдәҶ LR зҡ„зј©ж”ҫгҖӮ

    Page 13: Cerebras-GPT е®һзҺ°з»ҶиҠӮ¶

    • иЎЁж ј: еұ•зӨәдәҶе…·дҪ“зҡ„е®һзҺ°еҖјгҖӮ
    • Attention Scale дҪҝз”Ё \(1/D\)пјҲйҖҡеёёжҳҜ \(1/\sqrt{D}\)пјүгҖӮиҝҷдёҖж”№еҸҳеңЁе®ҪеәҰеўһеҠ ж—¶е°ӨдёәйҮҚиҰҒгҖӮ

    Page 14: muP зҡ„жӣҙж·ұеұӮз»ҶиҠӮ¶

    • иЎЁж ј: иҜҰз»ҶеҲ—еҮәдәҶжҜҸдёҖзұ»еҸӮж•°пјҲEmbedding, Attention, MLP, Logitsпјүзҡ„еҲқе§ӢеҢ–ж–№е·®е’ҢеӯҰд№ зҺҮзј©ж”ҫи§„еҲҷгҖӮ
    • жіЁж„Ҹ: Embedding еұӮе’Ң Output еұӮйҖҡеёёжңүзү№ж®Ҡзҡ„зј©ж”ҫи§„еҲҷпјҲеҰӮ \(1/M\) vs \(1/M^2\)пјүпјҢд»ҘдҝқиҜҒиҫ“е…Ҙиҫ“еҮәзҡ„зЁіе®ҡжҖ§гҖӮ

    Page 15: жү©еұ•еҚҸи®® (Scaling Protocol)¶

    • иЎЁж ј: Cerebras-GPT и®ӯз»ғзҡ„жЁЎеһӢеҲ—иЎЁгҖӮ
    • еұҖйҷҗжҖ§: иҜҘе·ҘдҪңеҸӘжҺўзҙўдәҶе®ҪеәҰ (Width) зҡ„жү©еұ•пјҢж·ұеәҰзӣёеҜ№еӣәе®ҡгҖӮиҝҷжҳҜ muP зҗҶи®әзӣ®еүҚдё»иҰҒиҰҶзӣ–зҡ„йўҶеҹҹгҖӮ

    Page 16: йӘҢиҜҒ muP (Replicating muP)¶

    • й—®йўҳ: muP зңҹзҡ„еғҸе®Јз§°зҡ„йӮЈж ·жңүж•Ҳеҗ—пјҹеҚівҖңжңҖдҪіеӯҰд№ зҺҮжҳҜеёёж•°вҖқпјҹ
    • йӘҢиҜҒ: иЎЁж јжҳҫзӨәпјҢеңЁ Baseline (SP) дёӢпјҢйҡҸзқҖе®ҪеәҰд»Һ 128 еҲ° 2048пјҢжңҖдҪі LR д»Һ 3.695 йҷҚеҲ°дәҶ 2.511гҖӮдҪҶеңЁ muP дёӢпјҢжңҖдҪі Base LR зЁіе®ҡеңЁ 3.7 е·ҰеҸігҖӮйӘҢиҜҒжҲҗеҠҹгҖӮ

    Page 17: muP зҡ„йІҒжЈ’жҖ§ (Robustness)¶

    • иғҢжҷҜ: зҺ°д»Ј LLM жңүеҫҲеӨҡз»„д»¶пјҲеҰӮ SwiGLU, RMSNormпјүе№¶дёҚеңЁ muP жңҖеҲқзҡ„зҗҶи®әеҒҮи®ҫдёӯгҖӮmuP иҝҳиғҪз”Ёеҗ—пјҹ
    • еҲ—иЎЁ:
    • SwiGLU / Squared ReLU: йІҒжЈ’гҖӮ
    • Batch Size: йІҒжЈ’гҖӮ
    • Initialization (Zero Query): йІҒжЈ’гҖӮ
    • RMSNorm Gains: дёҚйІҒжЈ’ (Break)гҖӮ
    • Lion Optimizer: дёҚйІҒжЈ’гҖӮ
    • Weight Decay: дёҚйІҒжЈ’гҖӮ

    Page 18: йІҒжЈ’жҖ§ - йқһзәҝжҖ§жҝҖжҙ»¶

    • з»“и®ә: ж— и®әжҳҜ SwiGLU иҝҳжҳҜ Squared ReLUпјҢmuP йғҪиғҪдҝқжҢҒжңҖдҪі LR зҡ„зЁіе®ҡжҖ§гҖӮ

    Page 19: йІҒжЈ’жҖ§ - Batch Size¶

    • з»“и®ә: Batch Size зҡ„еҸҳеҢ–дёҚеҪұе“Қ muP зҡ„ LR иҝҒз§»и§„еҫӢгҖӮиҷҪз„¶еҺҹе§ӢжҺЁеҜјжІЎиҖғиҷ‘ Batch SizeпјҢдҪҶз»ҸйӘҢдёҠжҳҜе…је®№зҡ„гҖӮ

    Page 20: йІҒжЈ’жҖ§ - еҲқе§ӢеҢ–¶

    • з»“и®ә: дҪҝз”Ё Zero Query InitпјҲе°Ҷ Query еҲқе§ӢеҢ–дёә 0пјүзӯүжҠҖе·§д№ҹдёҚеҪұе“Қ muPгҖӮ

    Page 21: дёҚйІҒжЈ’ - RMSNorm Gain¶

    • зҺ°иұЎ: еҰӮжһң RMSNorm еёҰжңүеҸҜеӯҰд№ зҡ„еўһзӣҠеҸӮж•°пјҲGain \(\gamma\)пјүпјҢmuP еӨұж•ҲгҖӮ
    • и§ЈеҶіж–№жЎҲ: зӣҙжҺҘз§»йҷӨ RMSNorm зҡ„еҸҜеӯҰд№ еҸӮж•°пјҲFix \(\gamma=1\)пјүгҖӮиҝҷеҮ д№ҺдёҚеҪұе“ҚжҖ§иғҪпјҢдҪҶжҒўеӨҚдәҶ muP зҡ„зү№жҖ§гҖӮ

    Page 22: дёҚйІҒжЈ’ - ејӮжһ„дјҳеҢ–еҷЁ (Lion)¶

    • зҺ°иұЎ: Lion дјҳеҢ–еҷЁеҹәдәҺз¬ҰеҸ·пјҲSignпјүпјҢдёҚз¬ҰеҗҲ muP зҡ„еҒҮи®ҫгҖӮж— жі•зӣҙжҺҘиҝҒз§»гҖӮ

    Page 23: дёҚйІҒжЈ’ - ејәжқғйҮҚиЎ°еҮҸ¶

    • зҺ°иұЎ: еҪ“ Weight Decay и®ҫдёә 0.1пјҲзҺ°д»Јж ҮеҮҶпјүж—¶пјҢmuP еӨұж•ҲгҖӮиҝҷжҳҜдёҖдёӘдё»иҰҒзҹӯжқҝгҖӮ

    Page 24: muP жҖ»з»“¶

    • жңүз”Ёеҗ—пјҹ жҖ»дҪ“йқһеёёжңүз”ЁгҖӮзү№еҲ«жҳҜеҜ№дәҺ SPпјҲж ҮеҮҶеҸӮж•°еҢ–пјүдёҚзЁіе®ҡзҡ„еңәжҷҜгҖӮ
    • е»әи®®: еҰӮжһңд»ҺеӨҙи®ӯз»ғпјҢдҪҝз”Ё muP еҲқе§ӢеҢ–е’ҢеҸӮж•°еҢ–еҸҜд»ҘиҠӮзңҒеӨ§йҮҸзҡ„и°ғеҸӮж—¶й—ҙгҖӮ

    Page 27: WSD еӯҰд№ зҺҮи°ғеәҰ (WSD Scheduler)¶

    • еӣҫиЎЁ (Figure 15): еҜ№жҜ”дәҶ Cosine и°ғеәҰпјҲй»„иүІпјүе’Ң WSD и°ғеәҰпјҲз»ҝиүІпјүгҖӮ

    • дјҳеҠҝ: еңЁ Stable йҳ¶ж®өпјҢдҪ еҸҜд»ҘйҡҸж—¶еҶіе®ҡвҖңз»“жқҹи®ӯз»ғвҖқгҖӮеҸӘйңҖд»ҺеҪ“еүҚ Checkpoint ејҖе§ӢеҒҡ DecayпјҢе°ұиғҪеҫ—еҲ°ж”¶ж•ӣзҡ„жЁЎеһӢгҖӮиҝҷе…Ғи®ёдёҖж¬Ўи®ӯз»ғжөӢиҜ•еӨҡдёӘж•°жҚ®йҮҸгҖӮ
