    显示人类可读的时间格式，包含 ​​秒 + 毫秒​​（如 2023-04-15 09:18:47.123）:
    ```bash
    journalctl -o short-precise
    ```

    示例输出​​：
    ```text
    Apr 15 09:18:47.123 hostname kernel: XFS (sda8): EXPERIMENTAL big timestamp...
    ```
