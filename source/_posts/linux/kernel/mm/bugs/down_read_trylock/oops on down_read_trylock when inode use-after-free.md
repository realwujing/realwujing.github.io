## 初步排查

```bash
gdb vmlinux
p ((struct inode *)0)->i_rwsem
Cannot access memory at address 0xa0
```

inode 虚拟地址为: `0x00000000000000a0` - `0xa0` = 0x0000000000000000`，说明inode是null指针，这种情况倾向于inode use-after-free。
